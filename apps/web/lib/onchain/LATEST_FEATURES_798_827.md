# Latest Onchain Features (798-827)

This document describes the 30 new onchain features added to the airdrop checker.

## Features Overview

### Timelock & Governance (798-800)
- **Timelock Queue Tracker**: Track timelock queue and execution delays
- **Proxy Upgrade Monitor**: Monitor proxy contract upgrades and implementations
- **Account Abstraction Analyzer**: Analyze account abstraction usage patterns

### Wallet Detection (801-802)
- **Smart Contract Wallet Detector**: Detect smart contract wallet deployments
- **Layer 2 Bridge Monitor**: Monitor Layer 2 bridge activity and efficiency

### Contract Analysis (803-813)
- **DEX Liquidity Scanner**: Scan liquidity across multiple DEXes
- **Contract Interaction Counter**: Count contract interactions
- **Event Listener Setup**: Setup event listeners for contracts
- **Contract Bytecode Diff**: Compare bytecode between contracts
- **Contract Deployment Cost**: Calculate deployment costs
- **Contract Function Decoder**: Decode function calls
- **Contract Function Selector**: Get function selectors
- **Contract Function Call Simulator**: Simulate function calls
- **Contract Event Indexer**: Index contract events
- **Contract ABI Validator**: Validate contract ABIs
- **Contract State Change**: Track contract state changes

### Multisig & Price Feeds (814-817)
- **Multisig Threshold**: Get multisig threshold requirements
- **Price Feed Validator**: Validate price feed data
- **Price Oracle Aggregator**: Aggregate prices from multiple oracles
- **Supply Change Notifier**: Monitor supply changes

### Distribution & Locks (818-820)
- **Distribution Snapshot**: Generate distribution snapshots
- **Lock Monitor**: Monitor token locks
- **Holder Segmentation**: Segment holders by balance

### Holder Analytics (821-827)
- **Holder Health**: Calculate holder health metrics
- **Holder Momentum**: Calculate holder momentum scores
- **Holder Quality**: Calculate holder quality scores
- **Holder Stability**: Calculate holder stability metrics
- **Holder Behavior**: Analyze holder behavior patterns
- **Holder Change**: Track holder changes over time
- **Holder Acquisition**: Track new holder acquisition

## API Endpoints

All endpoints are available at `/api/onchain/[feature-name]` with Reown Wallet integration.
